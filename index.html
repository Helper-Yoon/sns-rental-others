💡 헬로비전 수수료 계산 방식

헬로비전은 특별한 수수료 계산 방식을 사용합니다:
수수료 = 정상렌탈료 × 36개월 × 0.22 ÷ 1.1
다른 렌탈사와 달리 원본 수수료 없이 렌탈료 기준으로 직접 계산합니다.

지원 파일:
• 헬로비전_지엘_렌탈상품팀_상품리스트
• 헬로비전_코맨_헬로비전최신본
• 통합(일반,직영) 시트 자동 처리
• 1,000개 이상 상품 데이터 처리 가능

🚀 빠른 사용법

  1. 파일 업로드: 업로드 탭에서 엑셀 파일을 드래그 앤 드롭
  2. 자동 처리: 파일 유형을 자동 인식하여 적절한 계산 방식 적용
  3. 검색: 모델명, 상품명, 렌탈사명으로 즉시 검색
  4. 수수료 확인: 계산된 수수료가 카드에 표시됨
  5. 설정 변경: 관리 탭에서 계산 값 수정 가능

⚠️ 헬로비전 파일 처리 안내

헬로비전_지엘 & 헬로비전_코맨 파일:
• 통합(일반,직영) 시트를 자동으로 처리합니다
• 정상렌탈료 컬럼을 자동으로 찾아 계산합니다
• 1,000개 이상의 상품도 문제없이 처리됩니다
• EQP ID, 모델코드, 모델명 등 모든 정보를 추출합니다

팁: 헬로비전 파일은 복잡한 구조를 가지고 있지만, 시스템이 자동으로 헤더를 인식하고 데이터를 추출하므로 걱정하지 마세요!
