🎯 파싱 엔진 특징

  • 지능형 구조 감지: 각 시트의 헤더와 데이터 구조를 자동으로 인식
  • 다중 개월 처리: 하나의 상품을 여러 개월별로 자동 분할
  • 수수료 자동 계산: VAT 포함/미포함에 따른 정확한 수수료 계산
  • 대량 데이터 처리: 수천 개 상품도 빠르게 처리
  • 오류 방지: 잘못된 데이터 자동 필터링 및 검증
