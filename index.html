💡 파일 처리 방식

• 각 시트별로 자동 분석 및 렌탈사 인식
• 헤더 위치 자동 탐색 (1~30행 내)
• 모델명, 상품명, 렌탈료, 수수료 자동 추출
• 계산된 수수료 = 원본수수료 ÷ 1.1
• 중복 데이터 자동 업데이트

⚠️ 파일이 인식되지 않는 경우:
브라우저 콘솔(F12)에서 상세 로그를 확인하세요.

🚀 빠른 사용법

  1. 파일 업로드: 업로드 탭에서 엑셀 파일을 드래그 앤 드롭
  2. 자동 처리: 시트별로 자동 인식하여 적절한 계산 방식 적용
  3. 검색: 모델명, 상품명, 렌탈사명으로 즉시 검색
  4. 수수료 확인: 계산된 수수료가 카드에 표시됨
  5. 설정 변경: 관리 탭에서 계산 값 수정 가능

📊 자동 파싱 기능

대형가전_RH & 대형가전_코맨 파일:
• 모든 시트를 자동으로 처리합니다
• BS렌탈, 스마트렌탈, 캐리어 등 렌탈사별 구조 자동 인식
• 헤더 위치를 자동으로 찾아 데이터 추출
• 1,000개 이상의 상품도 안정적으로 처리

팁: 파일명이나 시트명에 렌탈사명이 포함되어 있으면 자동으로 인식하여 분류합니다!
